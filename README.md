# Passoia E-Commerce - Projeto Final Bloco 01

Sistema de gestão de estoque para e-commerce de cosméticos e medicamentos, focado em robustez e experiência do usuário.

## Funcionalidades Implementadas
O projeto segue a arquitetura **MVC** e aplica conceitos de **POO (Programação Orientada a Objetos)**.

- **CRUD Completo**: Listagem, consulta por ID, cadastro, atualização e exclusão de produtos.
- **Diferencial de Checkout**: 
  - Cálculo de frete dinâmico (Entrega vs. Retirada).
  - Lógica de desconto (5% OFF para pagamentos via PIX).
  - Recibo detalhado no console.
- **Validações de Segurança**:
  - Bloqueio de preços negativos ou zerados.
  - Validação de comprimento mínimo para nomes de produtos.
  - Tratamento de entradas de dados inválidas.
